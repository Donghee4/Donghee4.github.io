#### 실버4, 자료구조, 스택

```py
import sys
I=sys.stdin.readline

def sol():
    a = int(I())
    stack = []
    ans = []
    for i in range(a):
        ins = I().split()
        if len(ins) == 2:
            stack.append(ins[1])
        elif ins[0] == 'pop':
            if len(stack) == 0:
                ans.append('-1')
            else:
                ans.append(str(stack[-1]))
                del stack[-1]
        elif ins[0] == 'size':
            ans.append(str(len(stack)))
        elif ins[0] == 'empty':
            if len(stack) == 0:
                ans.append('1')
            else:
                ans.append('0')
        else:
            if len(stack) == 0:
                ans.append('-1')
            else:
                ans.append(str(stack[-1]))
    print('\n'.join(ans))
sol()
```
평범하게 리스트로 구현하였다.<br>
다른 코드를 살펴보니 나처럼 간단하게 구현한 사람도 있고, 명령어를 만들어서 사전에 저장하는 코드도 있었다.<br>
후자는 개인적으로 공부하기위해 기록함<br>

```py
import sys

stack = list()

command = dict(
    push=lambda x: stack.append(x),
    pop=lambda: stack.pop() if stack else -1,
    size=lambda: len(stack),
    empty=lambda: 0 if stack else 1,
    top=lambda: stack[-1] if stack else -1
    )

in_data = sys.stdin.readlines()
for c in in_data[1:]:
    c = c.strip()
    if c[:2] == 'pu':
        command['push'](c.split()[1])
    else:
        print(command[c]())
```
