#### 실버 1, 수학, dp, 정수론

```py
a = int(input())

if a == 1:
    print(0)
else:
    print((2 * (3**(a-2))) % ((10**9) + 9))
```

어떻게 풀어야 할지 고민이 많았지만,<br>
가장 빨리 푸는 방법은 예제를 보고 규칙을 찾는 방법이다.<br>
2일때 값을 찾고 규칙을 찾아보면,<br>
1: 0, 2: 2, 3:6, 4:13122 이므로 3씩 곱해진다는 규칙을 발견할 수 있다.<br>
정답은 맞지만, 이렇게 풀도록 의도된 건지는 알 수 없다.<br>
