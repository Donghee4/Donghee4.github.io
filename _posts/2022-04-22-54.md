#### 실버 2, 자료구조, 우선순위 큐

```py
import sys, heapq
a = list(map(int,sys.stdin.read().split()))
que = []

for i in a[1:]:
    if i == 0:
        if len(que) == 0:
            print(0)
        else:
            print(heapq.heappop(que)[1])

    else:
        heapq.heappush(que, (i if i>0 else abs(i)-0.5, i))
```
11279 최대 힙 문제와 비슷한 문제.<br>
코드도 거의 같지만 우선순위를 설정하는 부분에서 음수와 양수에 차이를 줬다.<br>
