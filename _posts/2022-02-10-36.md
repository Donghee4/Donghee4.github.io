#### 실버5, 문자열

```py
a = input()
b = input()
new =[]
c = '0123456789'
for i in range(len(a)): #숫자를 지운 뒤 new에 할당
    if a[i] not in c:
        new.append(a[i])

new = ''.join(new)  #리스트를 문자열로 변환

if b in new:
    print(1)
else:
    print(0)
```
입력받은 값 a에서 숫자를 지우기 위해 문자만 따로 리스트에 할당하였다.<br>

숫자를 지우는 또다른 방법:
```py
for i in c:
    a = a.replace(i,"") 
```
이렇게 replace를 이용해 숫자 부분을 지워줄 수 있다.<br>
입력값이 길지 않아 두 방법에서 속도차이는 거의 없다. <br>
